#### **1. 选择排序（以升序为例）**

- **步骤示例**：数组 `[5, 3, 8, 4, 6]`
    1. **第一轮**：在 `[5,3,8,4,6]` 中找到最小值 `3`，与第一个元素 `5` 交换 → `[3,5,8,4,6]`
    2. **第二轮**：在 `[5,8,4,6]` 中找到最小值 `4`，与第二个元素 `5` 交换 → `[3,4,8,5,6]`
    3. **第三轮**：在 `[8,5,6]` 中找到最小值 `5`，与第三个元素 `8` 交换 → `[3,4,5,8,6]`
    4. **第四轮**：在 `[8,6]` 中找到最小值 `6`，与第四个元素 `8` 交换 → `[3,4,5,6,8]`
- **核心逻辑**：每次确定一个位置的元素，通过 “选择 + 交换” 完成排序。

#### **2. 插入排序（以升序为例）**

- **步骤示例**：数组 `[5, 3, 8, 4, 6]`
    1. **初始**：已排序部分为 `[5]`，未排序部分为 `[3,8,4,6]`
    2. **第一轮**：取 `3`，插入到 `5` 前面 → 已排序 `[3,5]`
    3. **第二轮**：取 `8`，比 `5` 大，直接接在末尾 → 已排序 `[3,5,8]`
    4. **第三轮**：取 `4`，找到插入位置（`3` 和 `5` 之间），将 `5`、`8` 后移 → 已排序 `[3,4,5,8]`
    5. **第四轮**：取 `6`，插入到 `5` 和 `8` 之间 → 最终 `[3,4,5,6,8]`
- **核心逻辑**：已排序部分始终保持有序，新元素通过 “比较 + 后移” 找到合适位置插入。
